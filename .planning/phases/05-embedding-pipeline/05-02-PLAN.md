1. **Imports** — Use `createRequire` for `umap-js` and `gray-matter` (both CJS-only). Import `Ollama` from `ollama` normally. Use Node.js built-ins only (`node:fs`, `node:path`, `node:crypto`, `node:module`, `node:url`).

4. **`readDocs(rootDir)`** — Export. Iterate `CONTENT_DIRS`, read every `.md` file via `readFileSync`. Parse with `matter(raw)`. Return array of `{ id, category, filePath, relPath, title, tags, content, raw }`. Skip directories that don't exist (`existsSync` check). Accept `rootDir` param (defaults to `ROOT`) for testability.

5. **`loadCache(cachePath)`** — Export. Returns `{ version: 1, entries: {} }` if file missing. Otherwise `JSON.parse(readFileSync(cachePath, 'utf8'))`. Accept `cachePath` param (defaults to `CACHE_PATH`) for testability.

6. **`getVectors(docs, cache, ollama)`** — Export async. For each doc: compute `hash = createHash('sha256').update(doc.raw).digest('hex').slice(0, 16)`. If `cache.entries[doc.relPath]?.hash === hash`, push `cache.entries[doc.relPath].vector` (cache hit). Otherwise: build `text = doc.title + '\n\n' + doc.tags.join(' ') + '\n\n' + doc.content`, call `await ollama.embed({ model: MODEL, input: text })`, push `res.embeddings[0]`, update `cache.entries[doc.relPath] = { hash, vector }`. Wrap ollama call in try/catch for model-not-found error. Log `[cache hit]` or `[embedding]` for each doc.

7. **`reduce(vectors)`** — Export. Clamp `nNeighbors = Math.min(15, vectors.length - 1)`. Construct `new UMAP({ nComponents: 2, nNeighbors, minDist: 0.1 })`. Call `umap.fit(vectors)`. Log `UMAP: N docs, nNeighbors=M`. Return the fit result (array of `[x, y]`).

8. **`atomicWriteJSON(targetPath, data)`** — Export. Write to `targetPath + '.tmp'`, then `renameSync`. Ensure parent dir exists (`mkdirSync(..., { recursive: true })`).

9. **`main()`** — Not exported. Orchestrates: `await assertOllamaRunning()` → `readDocs()` → `loadCache()` → `await getVectors(docs, cache, ollama)` → if all cache hits, log and exit early without touching embeddings.json → `atomicWriteJSON(CACHE_PATH, cache)` → `reduce(vectors)` → build output object → `atomicWriteJSON(OUTPUT_PATH, output)` → log summary.
