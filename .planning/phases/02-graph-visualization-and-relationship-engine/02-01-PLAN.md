<task type="auto" tdd="true">
  <name>Task 1: Create src/lib/graph.js with buildGraphData and getRelatedDocs</name>
  <files>src/lib/graph.js</files>
  <behavior>
    buildGraphData(allDocs) — allDocs is Array of {id, data:{title,tags}, category}:
    - Returns {nodes, edges}
    - nodes: [{id, title, tags, url, tagCount, color, cluster}] where url = `/${category}/${id}/`
    - edges: only pairs where sharedTags.length >= 2 — [{source, target, weight, sharedTags}]
    - Color assignment: count tag frequency across all docs; top tag gets '#00E5FF', second '#FF2D78', third '#39FF14'; each node colored by its first matching top tag; unmatched nodes get '#8888AA'

    getRelatedDocs(currentDoc, allDocs):
    - Excludes currentDoc (match by id)
    - Maps others to {doc, sharedTags, category}
    - Filters to sharedTags.length > 0
    - Sorts by sharedTags.length descending
    - Returns top 5
  </behavior>
  <action>
    Create src/lib/ directory and src/lib/graph.js. Plain JavaScript, no TypeScript, no Astro imports.

    buildGraphData(allDocs):
    1. Map allDocs to nodes: {id: doc.id, title: doc.data.title, tags: doc.data.tags, url: `/${doc.category}/${doc.id}/`, tagCount: doc.data.tags.length, color: '#8888AA', cluster: null}
    2. Build edges: O(n^2) over node pairs. shared = nodes[i].tags.filter(t => nodes[j].tags.includes(t)). Push {source: nodes[i].id, target: nodes[j].id, weight: shared.length, sharedTags: shared} only if shared.length >= 2.
    3. Tag frequency: build tagFreq = {} by iterating over all nodes' tags. Sort entries by count descending. Take top 3 as topTags.
    4. Color assignment: const clusterColors = ['#00E5FF', '#FF2D78', '#39FF14']. For each node, loop topTags; if node.tags.includes(topTags[i]), set node.color = clusterColors[i], node.cluster = topTags[i], break.
    5. Return {nodes, edges}.

    getRelatedDocs(currentDoc, allDocs):
    1. Filter out currentDoc by id.
    2. Map to {doc: other, sharedTags: currentDoc.data.tags.filter(t => other.data.tags.includes(t)), category: other.category}
    3. Filter sharedTags.length > 0.
    4. Sort by sharedTags.length descending.
    5. Slice 0-5.
    6. Return array.

    Export: export { buildGraphData, getRelatedDocs };
  </action>
