    **2. Tag buttons** — after dotPositions is built but before initial redraw():
    ```javascript
    // Build tag filter buttons from all unique tags in corpus
    const allTags = [...new Set(docs.flatMap(d => d.tags))].sort();
    const tagContainer = document.getElementById('map-tag-filter');
    for (const tag of allTags) {
      const btn = document.createElement('button');
      btn.className = 'tag-filter-btn';
      btn.textContent = tag;
      btn.addEventListener('click', () => {
        if (selectedTags.has(tag)) {
          selectedTags.delete(tag);
          btn.classList.remove('active');
        } else {
          selectedTags.add(tag);
          btn.classList.add('active');
        }
        redraw();
      });
      tagContainer.appendChild(btn);
    }
    ```

    **3. ANY/ALL toggle** — after tag buttons:
    ```javascript
    const anyAllBtn = document.getElementById('map-anyall-btn');
    anyAllBtn.addEventListener('click', () => {
      anyAllMode = anyAllMode === 'ANY' ? 'ALL' : 'ANY';
      anyAllBtn.textContent = anyAllMode;
      anyAllBtn.classList.toggle('any-mode', anyAllMode === 'ANY');
      anyAllBtn.classList.toggle('all-mode', anyAllMode === 'ALL');
      redraw();
    });
    ```

    **4. Fuse.js search** — after ANY/ALL toggle:
    ```javascript
    // Initialize Fuse index once — search on input
    const fuse = new Fuse(docs, {
      keys: ['title', 'tags'],
      threshold: 0.35,
      includeScore: false,
    });

    document.getElementById('map-search').addEventListener('input', (e) => {
      const q = e.target.value.trim();
      searchQuery = q;
      if (!q) {
        matchingDocIds = null; // null = no filter active (all pass)
      } else {
        matchingDocIds = new Set(fuse.search(q).map(r => r.item.id));
      }
      redraw();
    });
    ```

    CRITICAL: `matchingDocIds = null` means "no search filter active" (searchOpacity returns 1.0 for all docs). This is already handled correctly in the searchOpacity() function from Plan 02 which checks `if (matchingDocIds === null) return 1.0`.

    Pitfall to avoid: Do NOT use `matchingDocIds = new Set(docs.map(d => d.id))` for "no filter" — use null and the null check in searchOpacity(). This prevents the "empty Set = dim all docs" bug.
