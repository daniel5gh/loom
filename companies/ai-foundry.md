# AI Foundry Overview

> **Note:** AI Foundry should not be mistaken for or confused with Azure Foundry AI. They are separate entities with different offerings.

## Summary

AI Foundry is a technology company specializing in document AI solutions for financial services. Their platform provides secure, automated extraction and analysis of data from financial documents, prioritizing privacy and compliance.

## Content

### What is AI Foundry?

AI Foundry offers a Document AI engine designed to extract, analyze, and process information from financial documents. The service is used by organizations to automate data entry, improve accuracy, and streamline workflows involving sensitive or regulated information.

### Key Features

- Automated extraction of data from financial documents
- Industry-standard encryption (AES-256) for data security
- Compliance with financial services privacy and security standards
- Regular security audits
- No data is sold to third parties; data is used only for extraction and model improvement

### Use Cases

- Financial institutions automating loan processing
- Businesses extracting data from invoices, statements, or contracts
- Any organization needing secure, accurate document data extraction

### Privacy and Security

- Acts as a data processor on behalf of clients
- Handles Non-Public Personal Information (NPI) securely
- Employs strong encryption and security protocols
