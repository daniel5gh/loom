# n8n Overview

## Summary

n8n is an open-source workflow automation platform that enables businesses to connect applications, automate processes, and integrate AI into their operations. It offers secure, enterprise-ready deployment options and supports over 500 app integrations.

## Content

### What is n8n?

n8n allows users to design, deploy, and manage automated workflows that connect various apps and services. It can be run on-premises or in the cloud, and features advanced security, collaboration, and performance capabilities for enterprise use.

### Key Features

- 500+ app integrations for workflow automation
- On-premises and cloud deployment options
- Advanced security: SSO, SAML, LDAP, encrypted secrets, RBAC
- Collaboration: Git control, multi-user workflows, isolated environments
- Performance: audit logs, workflow history, external storage
- White-label and embeddable automation for customers

### Use Cases

- Automating business processes and data flows
- Integrating AI and external APIs into workflows
- Providing automation capabilities to end users via white-label solutions
