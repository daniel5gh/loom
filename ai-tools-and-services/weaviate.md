# Weaviate Overview

## Summary

Weaviate is an open-source vector database designed for AI-powered search, retrieval-augmented generation (RAG), and agentic workflows. It enables developers to build production-ready AI applications with contextual search, scalable architecture, and enterprise-grade deployment options.

## Content

### What is Weaviate?

Weaviate provides a unified platform for smart, contextual search across unstructured data, RAG, and agentic AI. It supports embeddings, ranking, auto-scaling, and integrates with popular AI frameworks and cloud providers.

### Key Features

- AI-powered search and hybrid search capabilities
- Retrieval-augmented generation (RAG) support
- Agentic AI and workflow orchestration
- Billion-scale, production-ready architecture
- Enterprise-ready deployment (cloud or on-premises)

### Use Cases

- Building AI-powered search and knowledge management systems
- Deploying RAG pipelines and document agents
- Scalable, secure vector data storage for AI applications
