# OpenRouter Overview

## Summary

OpenRouter is a platform for routing and managing requests to large language models (LLMs) across multiple providers. It enables users to prioritize fast LLMs, explore different providers, and customize routing logic for optimal performance and cost.

## Content

### What is OpenRouter?

OpenRouter acts as a meta-router for LLMs, allowing users to send requests to various AI providers and models through a unified interface. It supports customization of routing strategies and performance optimization.

### Key Features

- Route requests to multiple LLM providers
- Prioritize speed, cost, or other criteria
- Customize routing logic and auto-router settings
- SDK support for loading skills and integrating with applications

### Use Cases

- Organizations seeking to optimize LLM usage across providers
- Developers building applications that require flexible LLM access
- Cost and performance-sensitive AI deployments
