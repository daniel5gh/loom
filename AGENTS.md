# Agent Guidelines

## Project overview

For a comprehensive understanding of this project's purpose, goals, and requirements, please refer to:

- [PRD.md](PRD.md) - Product Requirements Document for the Documented Research Collection

## Document standards

All research documents added to this collection should follow the structure defined in:

- [TEMPLATE.md](TEMPLATE.md) - Standard markdown template with frontmatter

### Key requirements

- Use the TEMPLATE.md structure for all new documents
- Include frontmatter with title, date, and tags
- Maintain a logical, flat or lightly nested folder structure
- Keep documents simple and searchable
- Focus on supporting conclusions with evidence

## Content guidelines

- Use clear, precise language
- Include specific details and supporting evidence
- Organize content logically within each document
- Tag documents appropriately for discoverability
- Ensure all research is well-documented and backed by sources
